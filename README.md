# Coding Agent Commands

Small repository to store reusable command prompts for coding agents.

## Current commands

- `issue.md`: Workflow prompt for starting issue-based work. It handles issue/task input, branch naming, implementation guardrails, testing, and the draft PR publishing flow.
- `pr-comments.md`: Workflow prompt for processing PR feedback. It groups comments into actionable blocks, enforces approval gates, and defines how to resolve bot/human threads safely.
